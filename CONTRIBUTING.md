## 👥 Contributing

Thank you for your interest in contributing to the New Tab project!

Project is still in developing stage. Contributions are highly welcome!

## 🌿 Branch Flow

All developing happens in `dev` branch before being merged into `main` branch:
- `dev` branch — develop/unstable version
- `main` branch — published/stable version
