# New Tab v0.3.2
[![许可证: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](../LICENSE)
[![版本](https://img.shields.io/badge/version-0.3.2-blue)]()

现代化的开源新标签页，提供个性化浏览体验。

## ✨ 功能
- 可定制布局和主题
- 响应式设计
- 快捷访问
- 内置背景
- 快速搜索功能
- 格言
- 天气组件（即将推出！）
- 待办清单（即将推出！）

## 🚀 快速开始
1. 下载发布的源代码
2. 打开 `New-Tab.html` 文件
3. 开始使用个性化新标签页
