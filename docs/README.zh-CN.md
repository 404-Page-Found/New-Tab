# New Tab v0.4.1
[![许可证: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](../LICENSE)
[![版本](https://img.shields.io/badge/version-0.4.1-blue)]()

现代化的开源新标签页，提供个性化浏览体验。

## ✨ 功能
- 可定制布局和主题
- 响应式设计
- 快捷访问
- 内置背景
- 快速搜索功能
- 格言
- 待办清单
- 天气组件（即将推出！）

## 🚀 快速开始
1. 从`Edge 插件`或已发布的`.crx`文件下载扩展程序
2. 要从`.crx`文件安装扩展程序，请转到浏览器中的管理扩展程序页面，启用开发者模式，拖放或选择`加载已解压的文件`，然后选择`.crx`文件
3. 开始使用您的个性化新标签页
